# 🚆 AccompTrainy ✨

Accomptrainy is a high-fidelity mobile application designed to solve a critical challenge: university student safety during late-night commutes. Built entirely within Figma, this project demonstrates an end-to-end UI/UX process focused on providing real-time accompaniment and peace of mind for students navigating campus and urban environments after dark.

This repository hosts the interactive prototype links and a rigorous evaluation report that explores how interaction design can reduce cognitive load in high-stress situations. From initial user research to iterative prototyping and user testing, the project showcases a deep commitment to accessible safety features and the development of seamless, "safety-first" interface patterns.
